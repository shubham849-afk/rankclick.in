## 📋 Deployment Steps (3 minutes)

### Step 1: Get GitHub Personal Access Token
1. Open: https://github.com/settings/tokens/new
2. Name: "Vercel Deploy"
3. Check: `repo` and `workflow`
4. Click "Generate token"
5. **Copy the token** (you won't see it again!)

### Step 2: Push Code to GitHub
```bash
cd c:\Users\shubh\Documents\clickrank.in
git push -u origin master
```
When asked for password, paste your token (not your GitHub password)

### Step 3: Deploy to Vercel
1. Go to https://vercel.com/dashboard
2. Click "Add New" → "Project"
3. "Import Git Repository"
4. Find and select `shubham849-afk/clickrank.in`
5. Click "Import"
6. Vercel will auto-deploy automatically ✨
